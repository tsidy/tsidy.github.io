Mon procédé

1. Analyse des besoins

Comprendre

Dans un premier temps, j’établis une Business request qui permet une meilleure compréhension des attentes. Dans une seconde partie, j’analyse les données sur lesquelles je travaille afin de retenir seulement celles dont j’ai besoin.

2. Requêtes et nettoyage

Préparation des données

À partir des données fournies (MySQL/ SQL Server/Excel/CSV/Python/ JSON), je nettoie les données en amont avant de les préparer. Pour cette étape, je transforme les informations à partir de la base de données, ou sur PowerQuery (code M), Excel, ou alors sur Python. Pour finir, je les importe sur l’outil BI demandé.

3. Création dashboard

Élaboration des visuelles

Après avoir réalisé les relations entre les différentes tables dans le modèle, et avoir créé les mesures intéressantes, je construis les différents visuels adaptés en fonction des destinataires (Clients/inter entreprise/ autres services). Création d’interactivité et/ou sous format mobile.

4. Communiquer

Transmettre aux métiers

Dans un dernier temps, je transmets mes analyses aux équipes métiers avec la rédaction d’ un rapport accompagnant mes visuels, ou en réalisant une présentation. Pour finir, je rédige une synthèse sur les attentes et objectifs, ainsi que sur les points clés à identifier.
